

		   PNG Alpha Viewer for Windows2000


1. はじめに

	世の中にはグラフィックビューワは沢山ありますが、半透明 PNG を
	ちゃんと半透明表示してくれるものはないようです。Windows2000 で
	はウィンドウの半透明表示を OS がサポートしてくれるのですから、
	半透明表示したいですよね？

2. 使い方

	使い方はいくつかあります。

	A) pngav.exe の引数にファイル名を指定して起動する。
	B) pngav.exe に画像ファイルを関連付けておく。
	C) pngav.exe を起動しておいて、画像ファイルをドロップする。

	沢山画像ファイルを指定したときは、スペースキー (又は N) で次の
	画像、バックスペースキー (又は P) で前の画像を見ることができま
	す。

	エスケープキー (又は Q) で終了します。

	Susie がインストールされているなら、勝手に Susie プラグインを
	使用します。インストールしていない場合は、pngav.exe と同じフォ
	ルダに Susie プラグイン 入れておくと勝手に使用します。

3. 手抜き

	ウィンドウの枠の形が少々変とか、タイトルバーの文字と右上の三つ
	のボタンが重なるとか、メニューがないとか、拡大さえできないとか、
	スクロールバーがないとか、Windows2000 以外では起動しないとか、
	いろいろありますが、あまり気にしないでください。画像ビューワを
	まじめに作る気がぜんぜんないので、なおす気はほとんどありません。
	むしろ有名どころのビューワがこのソフトに影響を受けて半透明表示
	を実装してくれるのを期待しています。
